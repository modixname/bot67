# Admin Bot Setup Guide

Этот бот получает логи от основного бота и показывает кто что писал.

## Настройка

### 1. Создайте второго бота в Telegram

1. Напишите @BotFather
2. Отправьте `/newbot`
3. Назовите бота (например: "Admin Logger")
4. Получите токен (например: `123456789:ABCdef...`)

### 2. Узнайте свой Chat ID

1. Напишите @userinfobot
2. Отправьте `/start`
3. Скопируйте ваш Chat ID (например: `123456789`)

### 3. Создайте файл .env.admin

Создайте файл `.env.admin` в корне проекта:

```env
ADMIN_BOT_TOKEN=8509724125:AAE7jF0cnM5cfNud7xwO4OWKBbHRbweqTlQ # Токен от BotFather
ADMIN_CHAT_ID=8336413944              # Ваш Chat ID
```

### 4. Запустите оба бота

**Основной бот:**
```bash
python3 bot.py
```

**Admin бот:**
```bash
python3 admin_bot.py
```

## Команды Admin бота

- `/start` - запуск бота
- `/logs` - показать последние 10 логов
- `/stats` - статистика (уникальные пользователи, сообщения)
- `/clear` - очистить все логи

## Что логируется

- Все текстовые сообщения (с указанием раздела)
- Нажатия на кнопки (callback)
- Отправка фото
- Команда /start

## Пример лога

```
📩 Новое сообщение:

👤 Пользователь: John
🆔 ID: 123456789
💬 Сообщение: [ai] Привет, как дела?
⏰ Время: 14:30:25
```

## Важно

- Admin бот работает независимо от основного
- Логи хранятся в памяти (после перезапуска теряются)
- Если admin бот не запущен - основной бот продолжает работать без отправки логов
- Не коммитьте `.env.admin` в git (он уже в .gitignore)
